## はじめに — 「デザインはいいけど、集客が...」

フリーランスでWeb制作をしていると、こんな場面に出くわしませんか？

- 納品後に「検索で出てこないんだけど」と言われる
- リニューアル案件の提案で、何を改善すべきか具体的に示せない
- 保守契約を提案したいが、毎月何をするのか説明しにくい

これらの課題は、**SEO診断を営業フローに組み込む**ことで解決できます。

この記事では、無料のSEO診断ツールを使って「課題の可視化 → 改善提案 → 保守契約」の流れを作る方法を紹介します。

## 1. SEO診断で「見えない課題」を可視化する

クライアントのサイトには、見た目ではわからないSEOの問題が潜んでいます。

- タイトルタグが長すぎる（または短すぎる）
- メタディスクリプションが未設定
- 構造化データ（JSON-LD）が入っていない
- OGPタグが設定されていない
- 見出し階層（h1〜h6）が崩れている

これらを手動でチェックするのは現実的ではありません。URLを入れるだけで45項目以上を自動チェックし、100点満点でスコア化できるツールがあると、**数字で課題を見せる**ことができます。

```
例：クライアントのサイトが62点だった場合

「現在のSEOスコアは62点です。
 構造化データが未設定で38点分が取れていません。
 これを改善すれば80点以上になり、
 Googleのリッチリザルトにも表示されやすくなります」
```

ポイントは**「何が問題か」と「どうすれば良くなるか」を同時に提示する**こと。診断結果をそのまま提案資料として使えます。

## 2. サイト全体診断で提案の説得力を上げる

1ページだけでなく、**サイト全体を一括診断**するとさらに効果的です。

ドメインを入力するだけで、sitemapに登録された全ページのSEOスコアを一括チェックできます。結果はスコアの低いページから順に表示されるので、**どのページを優先的に改善すべきか**がひと目でわかります。

```
提案例：

「御社のサイトを30ページ診断しました。
 平均スコアは68点ですが、以下の5ページが50点以下です：

 45点  /service/old-page
 48点  /blog/2023-post
 49点  /about
 52点  /contact
 55点  /products

 特に /service/old-page は構造化データとメタディスクリプションが
 未設定のため、この2つを追加するだけで70点以上に改善できます」
```

**「30ページ中5ページが問題」**という具体的な数字は、クライアントにとって非常にわかりやすい指標です。

## 3. 実際の提案フロー

### ステップ1：初回ヒアリング前にサイト全体を診断

商談前にクライアントのサイトをサイト全体診断にかけておきます。5分程度で完了します。

### ステップ2：スコアの低いページを3つピックアップ

全ページの結果から、改善効果が高そうなページを3つ選びます。トップページ、サービスページ、問い合わせページなど、コンバージョンに近いページを優先するのがコツです。

### ステップ3：改善コードを自動生成して提案に添える

診断結果から、具体的な改善コードが自動生成されます。

- JSON-LD構造化データ（FAQPage、Organization、Breadcrumb等）
- 最適化されたメタタグ
- OGPタグ
- llms.txt（AI検索対応）

**「こういうコードを追加すれば改善できます」と具体的なコードを見せる**ことで、「この人に頼めば実際に改善してくれる」という信頼感につながります。

### ステップ4：見積もりに改善作業を含める

```
見積もり例：

■ SEO改善パッケージ
- SEO診断・レポート作成          ¥0（無料ツールで実施）
- 構造化データ（JSON-LD）実装    ¥30,000
- メタタグ最適化（10ページ）     ¥20,000
- Core Web Vitals改善           ¥30,000
- 競合3サイトとの比較レポート    ¥10,000
                        合計     ¥90,000
```

診断自体は無料なので、**付加価値サービスとして提案**できます。

## 4. 保守契約につなげるスコア推移レポート

SEO改善は一度やって終わりではありません。これが**保守契約の根拠**になります。

### なぜ継続チェックが必要か

- CMSの更新でメタタグが消えることがある
- プラグインのアップデートで構造化データが壊れることがある
- Googleのアルゴリズム変更でスコアが変動する
- 新しいページを追加するたびにSEO設定が必要

### 月次レポートの作り方

毎月1回、主要ページのSEOスコアを個別チェックします。ダッシュボードにスコア推移グラフが自動生成されるので、それをスクリーンショットしてレポートに添付するだけです。

```
月次レポート例：

■ 2026年3月 SEOレポート
- トップページ：82点（前月比 +5）
- サービスページ：78点（前月比 +12）← 構造化データ追加の効果
- ブログ記事：65点（前月比 -3）← 要確認

改善実施：サービスページにFAQPage構造化データを追加
次月の施策：ブログ記事のメタディスクリプション最適化
```

**「前月比+12点」という数字**は、クライアントが保守費用の価値を実感できる最もわかりやすい指標です。

## 5. 競合比較で「なぜ今やるべきか」を伝える

「SEO対策は大事だと思うけど、今じゃなくてもいいかな...」

こういうクライアントには、**競合サイトとのスコア比較**が効果的です。

```
「御社のSEOスコアは62点ですが、
 競合のA社は85点、B社は78点です。
 特に構造化データの差が大きく、
 A社はリッチリザルトに表示されていますが、
 御社はされていません。
 この差を放置すると、検索からの流入で
 さらに差が開く可能性があります」
```

## 6. まとめ — SEO診断は「営業コスト0円の提案ツール」

SEO診断を営業フローに組み込むメリットをまとめます。

| フェーズ | やること | 効果 |
|---------|---------|------|
| 初回提案 | サイト全体診断 | 具体的な数字で課題を可視化 |
| 見積もり | 改善コード自動生成 | 技術力の証明＋工数削減 |
| 納品後 | 競合比較 | 「今やるべき理由」の提示 |
| 保守 | 月次スコアレポート | 継続契約の根拠 |

診断自体は無料で使えるので、**営業コスト0円で提案の質を上げられる**のがポイントです。

「デザインはできるけどSEOは苦手」というフリーランスこそ、ツールに頼るのが正解です。診断→改善コード生成まで自動化されていれば、SEOの専門知識がなくても具体的な改善提案ができます。
